# Schnider's Drums

Schnider's Drums - бесплатная онлайн школа барабанов

## Инициализация node_modules

В проекте используются фреймворки, к примеру, Bootstrap. Для корректного отображения элементов на странице, требуется инициализировать пакет следующим кодом в терминале вашей IDE:

```bash
yarn init
```
или

```bash
npm init
```
